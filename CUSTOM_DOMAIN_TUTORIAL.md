## 购买域名

### 什么是域名？

**简单理解：**

- 域名就是网站的地址，如：`baidu.com`、`taobao.com`
- 我们要购买一个自己的域名，如：`employee-system.com`
- 购买后，可以用这个域名访问你的网站

### 推荐的域名注册商

| 平台         | 价格        | 优点       | 推荐度   |
| ---------- | --------- | -------- | ----- |
| Cloudflare | ¥50-100/年 | 最便宜，配置简单 | ⭐⭐⭐⭐⭐ |
| 阿里云        | ¥50-100/年 | 国内知名，稳定  | ⭐⭐⭐⭐⭐ |
| 腾讯云        | ¥50-100/年 | 国内知名，稳定  | ⭐⭐⭐⭐  |
| GoDaddy    | ¥60-120/年 | 国际知名     | ⭐⭐⭐   |

### 详细购买步骤（以Cloudflare为例）

#### 步骤1：访问Cloudflare注册页面

1. 打开浏览器
2. 在地址栏输入：`https://www.cloudflare.com/products/registrar`
3. 按回车访问

**页面样子：**

```
┌─────────────────────────────────────────┐
│  Cloudflare Registrar                 │
│                                     │
│  [搜索域名]                          │
│                                     │
│  .com  .net  .org  .io         │
└─────────────────────────────────────────┘
```

#### 步骤2：搜索想要的域名

1. 在搜索框中输入你想要的域名
2. 例如：输入 `employee-system`
3. 点击搜索按钮（放大镜图标）

**搜索结果示例：**


**提示：**

- 如果想要的域名已被注册，尝试其他后缀（.net、.org、.cn）
- 或者换个名字（如：`my-employee-system.com`）

#### 步骤3：添加到购物车

1. 找到可用的域名
2. 点击"添加到购物车"或"Add to Cart"按钮
3. 确认购物车中有这个域名

#### 步骤4：结账购买

1. 点击购物车或"Checkout"按钮

## 在Cloudflare中配置

### 步骤1：访问Cloudflare Dashboard

1. 打开浏览器
2. 在地址栏输入：`https://dash.cloudflare.com`
3. 按回车访问

**如果未登录：**

1. 输入邮箱和密码
2. 点击"Sign In"或"登录"按钮

### 步骤2：添加站点

1. 在Cloudflare Dashboard中，点击 **Add a Site** 按钮
2. 输入你的域名：`employee-system.com`
3. 点击 **Add Site** 按钮

**添加站点的样子：**

```
┌─────────────────────────────────────────┐
│  Add a Site                          │
│                                     │
│  域名：[employee-system.com      ]    │
│                                     │
│  计划：Free（免费）                  │
│                                     │
│  [取消]  [添加站点]                 │
└─────────────────────────────────────────┘
```

**添加成功后：**

```
┌─────────────────────────────────────────┐
│  employee-system.com                   │
│                                     │
│  状态：Active                        │
│  计划：Free                         │
│                                     │
│  [进入DNS设置]                       │
└─────────────────────────────────────────┘
```

### 步骤3：配置DNS记录

1. 点击 **DNS** 标签
2. 点击 **Add Record** 按钮
3. 按照Vercel提供的信息填写

**DNS记录配置：**

**A. 基本信息**

```
类型：CNAME
名称：employee-system
目标：cname.vercel-dns.com
```

**B. 高级设置（可选）**

```
代理状态：已启用（Proxied）
TTL：Auto
```

**添加DNS记录的界面：**

```
┌─────────────────────────────────────────┐
│  Add DNS Record                      │
│                                     │
│  类型：[CNAME          ▼]           │
│  名称：[employee-system ]            │
│  目标：[cname.vercel-dns.com ]     │
│                                     │
│  代理状态：☑ 已启用               │
│  TTL：[Auto          ▼]             │
│                                     │
│  [取消]  [保存]                     │
└─────────────────────────────────────────┘
```

**点击保存后：**

```
┌─────────────────────────────────────────┐
│  DNS Records                          │
│                                     │
│  ✓ employee-system.com                 │
│    类型：CNAME                        │
│    目标：cname.vercel-dns.com          │
│    代理：☑ 已启用（橙色云朵）     │
│                                     │
│  [添加更多记录]                       │
└─────────────────────────────────────────┘
```

**重要提示：**

- 代理状态必须启用（橙色云朵图标）
- 这样才能使用Cloudflare的CDN加速
- 如果是灰色云朵，点击启用

### 步骤4：等待DNS生效

**DNS生效时间：**

- 通常：10分钟到1小时
- 最长：24小时
- 可以使用工具检查

**检查DNS是否生效：**

**方法一：使用在线工具**

1. 访问：https://www.nslookup.io
2. 输入：`employee-system.com`
3. 点击查询

**方法二：使用命令行（Windows）**

```powershell
nslookup employee-system.com
```

**方法三：使用命令行（Mac/Linux）**

```bash
dig employee-system.com
```

**生效后的结果：**

```
┌─────────────────────────────────────────┐
│  DNS查询结果                         │
│                                     │
│  域名：employee-system.com             │
│  CNAME：cname.vercel-dns.com          │
│  状态：✓ 已生效                  │
```

## 常见问题

### 问题1：域名购买失败

**症状：** 支付失败或购买失败

**解决：**

**A. 支付失败**

1. 检查银行卡余额
2. 尝试其他支付方式（支付宝、微信）
3. 联系客服

**B. 域名已被注册**

1. 尝试其他后缀（.net、.org、.cn）
2. 换个名字（如：`my-employee-system.com`）
3. 使用域名生成器找可用域名

---

### 问题2：Vercel添加域名失败

**症状：** 添加域名时显示错误

**解决：**

**A. 域名格式错误**

- 确保输入完整域名（如：`employee-system.com`）
- 不要只输入名称（如：`employee-system`）

**B. 域名已被其他Vercel项目使用**

- 确保域名没有被其他项目使用
- 如果被使用，先从其他项目移除

**C. DNS配置错误**

- 检查DNS记录是否正确
- 等待DNS生效后再验证

---

### 问题3：Cloudflare配置失败

**症状：** DNS记录添加失败

**解决：**

**A. 记录冲突**

- 检查是否已存在相同记录
- 删除旧记录后重新添加

**B. 值错误**

- 确保目标值正确：`cname.vercel-dns.com`
- 不要有多余的空格

**C. 代理状态未启用**

- 点击记录右侧的代理图标
- 确保显示为橙色云朵（已启用）

---

### 问题4：访问新域名失败

**症状：** 访问 `https://employee-system.com` 显示错误

**解决：**

**A. DNS未生效**

- 等待10分钟到24小时
- 使用nslookup工具检查

**B. Vercel未验证**

- 回到Vercel Dashboard
- 检查域名状态
- 如果显示错误，查看错误信息

**C. 浏览器缓存**

- 清除浏览器缓存
- 尝试无痕模式
- 换个浏览器试试

---

### 问题5：访问速度仍然慢

**症状：** 配置自定义域名后，速度仍然慢

**解决：**

**A. 检查Cloudflare代理**

- 确保DNS记录的代理已启用
- 应该是橙色云朵图标

**B. 清除缓存**

- 在Cloudflare中清除缓存
- 点击Caching → Purge Everything

**C. 检查网络**

- 尝试其他网络（手机热点）
- 检查是否有网络限制
